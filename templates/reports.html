Anomaly Reports

Save and view anomaly reports from video analysis. Start the pipeline on the Dashboard to record video and detect anomalies.

Recording & recognition: Go to Dashboard → start the pipeline for your camera(s). Video is recorded and employees are recognised automatically. Anomalies (idle time, unauthorized presence, etc.) are logged as events. To run on existing recordings (e.g. under data/recordings/Office_Corridor/): use the API POST /api/pipeline/start/{camera_id}?source_url=E:/path/to/recording.mp4 (see API docs or docs/RUN_RECORDINGS.md).

Person detections

Authorised (employees on shift) and unauthorised (unknown) persons detected, with captured images. Run the pipeline to capture new sightings.

What is “track #1, #2, #9”? During one pipeline run (one video or one live stream), each distinct person the system starts following gets a track number: track 1 = first person seen, track 2 = second, etc. So “Unknown (track #9)” means the 9th person (or 9th new appearance) in that run — the video is not loaded multiple times; it’s one run with multiple people detected.
Recording: For live cameras, one recording per camera is created; when you Stop the pipeline, that file is closed and moved to the archive folder automatically. When you analyse a recording file, the app reads it once and does not create a new recording.
